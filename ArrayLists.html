ArrayLists

A quick tutorial on the functions and usages of ArrayLists.

What is an ArrayList?
An ArrayList is a type of data structure which functions similarly to an Array, however several key differences are present. While an arrayâ€™s size cannot be changed once it has been set, an ArrayList increases its capacity once its current list is completely filled, and thus ArrayLists are resizable. Another key difference about ArrayLists is that they can only hold objects, and not primitive data type elements.
As seen in the diagram, ArrayLists can continuously increase or decrease their capacity to accommodate for new element additions, unlike Arrays.

Declaring and Initializing ArrayLists:
To initialize an ArrayList, the following syntax can be utilized:

ArrayList ArrayListName = new ArrayList();

ArrayList Methods:
The ArrayList class has several unique functions which make it an appealing option when storing data.
Assume an ArrayList named testList has been properly initialized.

Add Method (1 Parameter): To add data to an existing ArrayList, the add method can be used. ThisThis method will add the object to the end of the ArrayList, resulting in all following value being shifted one index higher. To use this method, the following syntax can be utilized: testList.add(E obj).

Add Method (2 Parameters): Data can also be added to a specific position. The first parameter of this method determines the position at which the object will be inserted at, and the second parameter refers to the object. To use this method, the following syntax can be utilized: testList.add(int insertionPoint, E obj);

Set Method: This method replaces an element value with another value. It also returns the element value that was previously stored at that index. To use this method, the following syntaxa can be utilized: testList.set(int replacementPoint, E obj);

Get Method: This method returns an element value at a defined index. To use this method, the following syntax can be utilized: testList.get(int index);

Remove Method: This method removes an element value at a defined index. This results in the next value being shifted one index lower. To use this method, the following syntax can be utilized: testList.remove(int index);
