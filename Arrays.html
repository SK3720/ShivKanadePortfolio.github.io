Arrays

A quick tutorial on the functions and usages of Arrays.

What is an Array?
An Array is a type of data structure that is able to hold multiple data values of the same type. Array’s are composed of “elements”, which simply refers to the position of a value in an Array. Each element in the array has a corresponding index, with the first element in an Array being located at index 0, the second element at index 1, and so on. In the example, an array of 5 elements is seen. At element 1, located at index 0, the value stored is 10. This array stores only integers, however arrays can be initialized to any data type (and will only be able to hold that type of data).

Declaring and Initializing an Array
The syntax to declaring an array is as follows:

arrayDataType[] arrayName;
arrayName = new arrayDataType[arraySize];>

An Array can also be initialized in one statement: arrayDataType[] arrayName = new arrayDataType[arraySize];

Once an array is created, space is allocated to contain the elements of the array. If a set of values is not provided during the initialization of an array, default values are allocated to the array’s elements, such as null values for Strings. To provide a list of values when creating an array, the following syntax can be utilized.

arrayDataType[] arrayName = {Value1, Value2, Value 3…};

Here, the number of values provided defines the array size. However, once an array’s size has been set, that length is fixed and hence cannot be changed. If a non existent element in an array is attempted to be accessed, such as index 5 of an Array of 5 elements, an ArrayIndexOutOfBoundsException is thrown.

What is a 2D Array?
A 2D Array is an array of arrays. Each element in this data structure can be referred to by a row index and a column index. A 2D array which is of size 3 rows and 5 columns can be seen on the right. This is a visual representation of how data is stored in a 2D array.

Declaring and Initializing a 2D Array
The syntax to declaring a 2D array is as follows

arrayDataType[][] arrayName;
arrayName = new arrayDataType[rowLength][columnLength];

An array can also be initialized in one statement: arrayDataType[][] arrayName = new arrayDataType[rowLength][columnLength];

Alternatively, the data for the 2D array can be store in the array during its initialization. The following syntax can be used to do so:

arrayDataType[][] arrayName ={ {Value1, Value2, Value3…}, {Value4, Value5, Value 6}...};

Note that similar to 1D array, the size of a 2D array cannot be changed once it has been set. In addition, if no data is allocated to element slots, default values are set which correspond to the dataType of the 2D array.
