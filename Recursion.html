Recursion

A quick tutorial on what Recursion is and how it functions.

What is Recursion?
Recursion occurs when a method invokes itself from within its arguments. Recursion can replace iterative methods, and can be more efficient and simple compared to iterative solutions. Recursive methods solve problems by returning a simplified version of the original problem until the base case is met, after which it returns the final solution.

Recursion Example:
In the example seen on the right, a solution to finding the sum of all numbers up to a given number is shown. As can be seen, the recursive solution is considerably less complex than the iterative solution. The for loop is completely replaced by the recursive statement. A base case is also present to ensure that infinite recursion does not occur, and and answer is returned.

Recursive Process Example The diagram seen represents the process a recursive method takes to calculate the factorial of a number, which is 5 in this case. As seen, the recursive process breaks down each step to finding the solution into an easier step. From the second last step, where from 2*1!, 1! returns 1, the base is reached. Then, this final return value is plugged into the step above it, from which the value of 2*1! is returned to the step above it, and so on till the final value of 120 is reached.
